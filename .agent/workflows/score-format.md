# Format Nhập Điểm Vi Phạm

## Cú pháp
```
Tuần X:<Tên/Mã>|<điểm>|<lí do>|<ngày>|
```

## Quy tắc
- **Tuần X**: Số tuần (ví dụ: Tuần 12)
- **Tên/Mã**: Tên học sinh hoặc mã số (ví dụ: 105016 hoặc Hồng Hà)
- **Điểm**: Điểm trừ (số âm, ví dụ: -0.5)
- **Lí do**: Lý do vi phạm (ví dụ: Trễ, Vắng KP, Không đồng phục)
- **Ngày**: Ngày vi phạm (dd/mm/yyyy, ví dụ: 14/01/2026)
- Nhiều học sinh cùng tuần phân cách bằng `;`

## Ví dụ
```
Tuần 12:105016|-0.5|Trễ|14/01/2026|;105030|-0.5|Trễ|14/01/2026|;105006|-0.5|Trễ|14/01/2026|
```

Nghĩa là:
- Mã 105016 (Hồng Hà): trừ 0.5 điểm vì Trễ ngày 14/01/2026
- Mã 105030 (Trà My): trừ 0.5 điểm vì Trễ ngày 14/01/2026
- Mã 105006 (Kim Anh): trừ 0.5 điểm vì Trễ ngày 14/01/2026
